### 1. Installation
1. Download and install **AutoHotkey v2** from [autohotkey.com](https://www.autohotkey.com/).
2. Download the latest release from this repository (or clone it).
3. Extract everything and double-click `XB_Redragon_Profile_Manager.exe` to run.

No admin rights needed ‚Äî fully portable! ü™∂

### 2. First-Time Setup (One time only)
1. Open your Redragon (or other) software.
2. In the tool, go to the **Settings** tab:
   - Click **Select Redragon EXE** ‚Üí choose your software's `.exe` (e.g., `OemDrv.exe`).
   - Click **Capture Import Button** ‚Üí hover your mouse over the "Import/Load" button in the software ‚Üí press **F8**.
   - Test it with **Test Import Click** ‚Äî it should click the button automatically.
3. Place your profile files (e.g., `.jmk`, `.jmr`) in the `Profiles` folder (or use **Set Profiles Folder** to choose any location).
4. In Settings, select your profile file extension (or `*.*` for all types).

You're ready! üöÄ

### 3. Adding a Game
1. Go to the **Game Editor** tab.
2. Enter a game name.
3. Click **Browse** next to "Game EXE Path" ‚Üí select the game's `.exe`.
4. Click **Browse** next to "Profile Path" ‚Üí select the profile you want for this game.
5. (Optional) Add a launch delay if the software needs extra time.
6. Click **Add / Update**.

### 4. Launching Games
- Go to the **Game Launcher** tab.
- Search or select your game.
- Click **Launch Game** ‚Äî your profile will auto-import and the game will start!

**Pro tip**: In Game Editor, select a game and click **Create Shortcut** ‚Üí a desktop shortcut is made that loads the profile automatically on double-click! üñ±Ô∏è

### 5. Managing Profiles
- Go to the **Profile Manager** tab.
- Search and select profiles.
- Add notes, import manually, or click **Backup Profile** for safety.
- Use **Set Profiles Folder** to load profiles from anywhere on your PC.

### 6. Customization
- In **Settings** ‚Üí click **Pick** under Colors to open the **Color Theme Picker**.
- Use sliders, or paste any HEX code (e.g., `#FFA500`) directly!
- Sorting and profile extensions are also configurable here.
